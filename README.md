# AirPulse 🌍

AI-powered Air Quality Prediction Dashboard built using Python, XGBoost, and Streamlit.

## Features

- AQI Prediction
- Air Quality Classification
- Weekly AQI Trend Forecast
- Interactive Dashboard

## Technologies Used

- Python
- Streamlit
- XGBoost
- Pandas
- Plotly
